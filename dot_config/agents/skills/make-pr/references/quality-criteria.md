# 프롬프트 품질 평가 기준

> 이 문서는 `/make-pr-with-prompts` 스킬이 프롬프트 분석 시 참조하는 평가 기준입니다.
> PR의 Prompt Journal에서 각 프롬프트를 A/B/C 등급으로 분류할 때 사용합니다.

---

## 등급 분류

### 🟢 A등급 — 모범 프롬프트

**정의**: 5가지 관점 합산 **8-10점** (높음=2, 중간=1, 낮음=0)

**특징**:
- 달성하려는 목표와 이유가 명확하게 서술됨
- 관련 파일 경로, 기존 코드, 제약 조건 등 충분한 컨텍스트 제공
- "~하면 완료" 같은 검증 가능한 결과 기준 포함
- 관련 문서, 예시, 참고 코드 등을 함께 제공

**예시** (Akira 프로젝트):
```
FR-01에 따라 Slack 메시지 수신 시 언어 감지 → 번역 → 스레드 게시 흐름을 구현해줘.
- 파일: src/handlers/message_handler.py
- 언어 감지는 langdetect 라이브러리 사용
- 한국어/일본어만 처리, 나머지는 무시
- 처리 후 로그에 "translated: {lang} -> {lang}" 형태로 기록
- 완료 기준: test_message_handler.py의 모든 테스트 통과
```

---

### 🟡 B등급 — 양호 프롬프트

**정의**: 5가지 관점 합산 **5-7점**

**특징**:
- 무엇을 하려는지는 이해할 수 있으나 세부 조건이 불명확
- 컨텍스트가 일부 빠져 있어 Agent가 추가 질문이 필요할 수 있음
- 검증 기준이 모호하거나 없음

**예시**:
```
message_handler.py에 언어 감지 기능 추가해줘.
한국어랑 일본어만 처리하면 되고, 다른 언어는 무시.
```

---

### 🔴 C등급 — 개선 필요 프롬프트

**정의**: 5가지 관점 합산 **0-4점**

**특징**:
- 무엇을 원하는지 파악하기 어려움
- 컨텍스트 없이 결과만 요구
- 여러 기능을 한꺼번에 요청
- "잘 만들어줘", "완성해줘" 같은 막연한 표현

**예시**:
```
번역 기능 만들어줘
```

## 5가지 평가 관점

### 1. 명확성 (Clarity)

| 수준 | 설명 |
|------|------|
| 높음 | 목표, 범위, 제약이 명확히 서술됨 |
| 중간 | 목표는 있으나 범위나 제약이 불명확 |
| 낮음 | 의도 파악 자체가 어려움 |

**체크포인트**: "이 프롬프트만 봐도 무엇을 해야 하는지 바로 알 수 있는가?"

---

### 2. 컨텍스트 제공 (Context)

| 수준 | 설명 |
|------|------|
| 높음 | 파일 경로, 기존 코드, 관련 FR, 제약 조건 모두 제공 |
| 중간 | 일부 컨텍스트 제공 (파일명은 있지만 코드 없음 등) |
| 낮음 | 컨텍스트 없음 |

**체크포인트**: "Agent가 추가 정보를 요청하지 않고 바로 작업할 수 있는가?"

---

### 3. 검증 가능성 (Verifiability)

| 수준 | 설명 |
|------|------|
| 높음 | "테스트 통과", "특정 출력값", "특정 동작" 등 명확한 완료 기준 |
| 중간 | 어느 정도 확인 가능하나 기준이 주관적 |
| 낮음 | 완료 기준 없음 |

**체크포인트**: "결과물이 기대를 충족했는지 어떻게 확인할 수 있는가?"

---

### 4. 효율성 (Efficiency)

| 수준 | 설명 |
|------|------|
| 높음 | 적절한 크기의 단일 작업 단위 |
| 중간 | 약간 크거나 작지만 처리 가능한 범위 |
| 낮음 | 너무 광범위하거나 (여러 기능 한꺼번에) 너무 세분화됨 |

**체크포인트**: "이 요청을 한 번의 대화 턴으로 완료할 수 있는가?"

---

### 5. 재사용성 (Reusability)

| 수준 | 설명 |
|------|------|
| 높음 | 비슷한 작업에 패턴으로 재사용 가능 |
| 중간 | 약간의 수정으로 재사용 가능 |
| 낮음 | 일회성, 패턴화하기 어려움 |

**체크포인트**: "이 프롬프트 구조를 팀원이 다른 작업에도 쓸 수 있는가?"

---

## 종합 등급 산정

각 관점의 수준(높음=2, 중간=1, 낮음=0)을 합산:

| 총점 | 등급 |
|------|------|
| 8-10점 | A등급 (모범) |
| 5-7점 | B등급 (양호) |
| 0-4점 | C등급 (개선 필요) |
