## 한국어와 일본어의 유사점 활용

### 어순이 같다 (SOV)

| 한국어 | 일본어 | 구조 |
|---|---|---|
| 나는 사과를 먹는다 | 私はりんごを食べる | 주어 + 목적어 + 동사 |
| 학교에 간다 | 学校に行く | 장소 + 에 + 동사 |

- 비유: 한국어 문장을 그대로 일본어 단어로 바꾸면 거의 맞는다!
- 한국어 실력이 일본어 학습의 최대 무기

### 한자어 공유

- 한국어와 일본어는 많은 한자어를 공유
- 비유: "같은 한자 집안의 사촌"

| 한국어 | 일본어 | 한자 | 발음 비교 |
|---|---|---|---|
| 도서관 | としょかん | 図書館 | 도서관 ≈ 토쇼칸 |
| 약속 | やくそく | 約束 | 약속 ≈ 야쿠소쿠 |
| 준비 | じゅんび | 準備 | 준비 ≈ 쥰비 |
| 가족 | かぞく | 家族 | 가족 ≈ 카조쿠 |
| 시간 | じかん | 時間 | 시간 ≈ 지칸 |
| 무료 | むりょう | 無料 | 무료 ≈ 무료- |

### 조사 체계가 비슷하다

- 한국어: 은/는, 이/가, 을/를, 에, 에서, 도
- 일본어: は, が, を, に, で, も
- 거의 1:1 대응 → 한국어 조사를 일본어로 치환하면 됨

## 히라가나/가타카나 기본 체계

### 히라가나 (ひらがな) - 일본어 고유 표기

#### 기본 50음도 (오십음도)

|  | あ단 | い단 | う단 | え단 | お단 |
|---|---|---|---|---|---|
| 모음 | あ(a) | い(i) | う(u) | え(e) | お(o) |
| か행 | か(ka) | き(ki) | く(ku) | け(ke) | こ(ko) |
| さ행 | さ(sa) | し(shi) | す(su) | せ(se) | そ(so) |
| た행 | た(ta) | ち(chi) | つ(tsu) | て(te) | と(to) |
| な행 | な(na) | に(ni) | ぬ(nu) | ね(ne) | の(no) |
| は행 | は(ha) | ひ(hi) | ふ(fu) | へ(he) | ほ(ho) |
| ま행 | ま(ma) | み(mi) | む(mu) | め(me) | も(mo) |
| や행 | や(ya) | - | ゆ(yu) | - | よ(yo) |
| ら행 | ら(ra) | り(ri) | る(ru) | れ(re) | ろ(ro) |
| わ행 | わ(wa) | - | - | - | を(wo) |
| | ん(n) | | | | |

#### 탁음 (だくおん) · 반탁음 (はんだくおん)

| 기본 | 탁음 | 반탁음 |
|---|---|---|
| か→が(ga) | さ→ざ(za) | は→ば(ba) / ぱ(pa) |
| き→ぎ(gi) | し→じ(ji) | ひ→び(bi) / ぴ(pi) |
| く→ぐ(gu) | す→ず(zu) | ふ→ぶ(bu) / ぷ(pu) |
| け→げ(ge) | せ→ぜ(ze) | へ→べ(be) / ぺ(pe) |
| こ→ご(go) | そ→ぞ(zo) | ほ→ぼ(bo) / ぽ(po) |

- 비유: 탁음은 "점 두 개(゛)를 찍으면 소리가 탁해지는 것", 반탁음은 "동그라미(゜)를 찍으면 ㅍ 소리"

### 가타카나 (カタカナ) - 외래어·의성어 표기

- 히라가나와 같은 발음, 다른 모양
- 사용처: 외래어(コーヒー coffee), 의성어(ドキドキ 두근두근), 강조
- 비유: 한국어의 "외래어 표기"와 같은 역할

### 문자 사용 구분

| 문자 | 용도 | 예시 |
|---|---|---|
| 히라가나 | 고유어, 문법 요소 | たべる (먹다), の (의) |
| 가타카나 | 외래어, 의성어, 강조 | コンピュータ (컴퓨터) |
| 한자 | 의미 전달 (명사, 동사 어간) | 食べる (먹다), 学校 (학교) |

## 한국어 화자가 자주 틀리는 포인트

### 1. は(wa) vs が(ga) 구분

- 가장 어려운 포인트 (한국어의 은/는 vs 이/가와 유사하지만 완전히 같지 않음)
- **は**: 주제 (이미 알고 있는 것, "~에 대해 말하자면")
  - 비유: "이미 알고 있는 배경 설정"
- **が**: 주격 (새로운 정보, 누가/무엇이)
  - 비유: "새로 등장하는 주인공 소개"
- 예시:
  - "**私は**学生です" (저**는** 학생입니다) — 자기소개 (주제)
  - "**誰が**来ましたか" (**누가** 왔습니까?) — 새 정보 질문

### 2. 자동사 vs 타동사 쌍

- 한국어는 "열다/열리다"처럼 형태가 비슷하지만, 일본어는 완전히 다른 단어인 경우 많음

| 타동사 (누가 ~을) | 자동사 (~이 저절로) | 의미 |
|---|---|---|
| 開ける(あける) | 開く(あく) | 열다/열리다 |
| 閉める(しめる) | 閉まる(しまる) | 닫다/닫히다 |
| 消す(けす) | 消える(きえる) | 끄다/꺼지다 |
| つける | つく | 켜다/켜지다 |
| 入れる(いれる) | 入る(はいる) | 넣다/들어가다 |

### 3. 敬語(경어) 과잉/부족 사용

- 한국어 존댓말과 비슷하지만, 일본어 경어는 3단계로 더 세분화
- 한국어 화자는 비즈니스 일본어에서 경어 수준 선택에 혼란을 겪음
- 자세한 내용은 [경어 체계](#경어-체계) 참조

### 4. 조사 혼동

| 자주 틀리는 패턴 | 올바른 표현 | 설명 |
|---|---|---|
| 学校**を**行く | 学校**に**行く | "~에 가다"는 に |
| バス**に**乗る (○) | - | "~에 타다"는 に (한국어와 동일) |
| 友達**に**会う (○) | - | "친구를 만나다"이지만 일본어는 に |
| 日本語**を**話せる | 日本語**が**話せる | 가능형은 が |

### 5. 장음·촉음 구분

- 장음: おばさん(아주머니) vs **おばあさん**(할머니) — 장단으로 의미 변화
- 촉음(っ): きて(와서) vs **きって**(잘라서/우표) — 짧은 멈춤으로 의미 변화
- 한국어에는 이런 구분이 없으므로 특히 주의

## 동사 활용 체계

### 동사 3그룹

| 그룹 | 특징 | 예시 |
|---|---|---|
| **1류 (5단동사)** | う단으로 끝남 | 書く, 話す, 読む, 飲む, 行く |
| **2류 (1단동사)** | いる/える로 끝남 | 食べる, 見る, 起きる, 寝る |
| **불규칙** | 2개만 존재 | する(하다), 来る(くる, 오다) |

- 비유: 1류는 "규칙적이지만 변화가 다양한 그룹", 2류는 "심플하게 る만 떼는 그룹"
- 주의: 모양은 2류 같지만 1류인 동사 → 帰る(かえる), 走る(はしる), 切る(きる), 知る(しる)

### 기본 활용표

#### 1류 동사 (예: 書く かく)

| 활용 | 형태 | 의미 |
|---|---|---|
| ます형 | 書きます | ~씁니다 (정중) |
| て형 | 書いて | ~쓰고, ~써서 |
| ない형 | 書かない | ~쓰지 않다 |
| た형 | 書いた | ~썼다 |
| 가능형 | 書ける | ~쓸 수 있다 |
| 의지형 | 書こう | ~쓰자 |
| 조건형 | 書けば | ~쓰면 |

#### 2류 동사 (예: 食べる たべる)

| 활용 | 형태 | 의미 |
|---|---|---|
| ます형 | 食べます | ~먹습니다 (정중) |
| て형 | 食べて | ~먹고, ~먹어서 |
| ない형 | 食べない | ~먹지 않다 |
| た형 | 食べた | ~먹었다 |
| 가능형 | 食べられる | ~먹을 수 있다 |
| 의지형 | 食べよう | ~먹자 |
| 조건형 | 食べれば | ~먹으면 |

### て형 변환 규칙 (1류 동사)

| 어미 | て형 | 예시 |
|---|---|---|
| く | いて | 書く → 書いて |
| ぐ | いで | 泳ぐ → 泳いで |
| す | して | 話す → 話して |
| む/ぶ/ぬ | んで | 読む → 読んで, 飛ぶ → 飛んで |
| る/う/つ | って | 帰る → 帰って, 買う → 買って |

- 예외: 行く → 行って (いって, 규칙대로면 いいて이지만 예외)

## 조사 비교

### 한국어-일본어 조사 대응표

| 한국어 | 일본어 | 예문 |
|---|---|---|
| ~은/는 (주제) | は | 私**は**学生です (저는 학생입니다) |
| ~이/가 (주격) | が | 雨**が**降る (비가 온다) |
| ~을/를 (목적격) | を | 本**を**読む (책을 읽다) |
| ~에 (장소·시간) | に | 学校**に**行く (학교에 가다) |
| ~에서 (장소·수단) | で | 図書館**で**勉強する (도서관에서 공부하다) |
| ~도 | も | 私**も**行く (나도 가다) |
| ~의 | の | 私**の**本 (나의 책) |
| ~와/과 | と | 友達**と**遊ぶ (친구와 놀다) |
| ~에게/한테 | に | 先生**に**聞く (선생님에게 묻다) |
| ~부터 | から | 9時**から**始まる (9시부터 시작) |
| ~까지 | まで | 5時**まで**働く (5시까지 일하다) |

### に vs で 구분

- **に**: 존재·도착점·시간 ("어디에 있다", "어디에 가다")
  - 비유: "목적지 핀 꽂기" 📍
- **で**: 동작 장소·수단 ("어디에서 하다", "뭘로 하다")
  - 비유: "활동 무대 설정" 🎬

## 경어 체계

### 3단계 경어

| 종류 | 의미 | 비유 |
|---|---|---|
| **丁寧語** (정중어) | ~です/~ます | 한국어 "~요/~습니다" |
| **尊敬語** (존경어) | 상대방 행동을 높임 | 한국어 "~하시다" |
| **謙譲語** (겸양어) | 자신을 낮춤 | 한국어 "~드리다" |

### 주요 동사 경어 변환

| 보통 | 존경어 | 겸양어 | 의미 |
|---|---|---|---|
| 行く/来る | いらっしゃる | 参る(まいる) | 가다/오다 |
| いる | いらっしゃる | おる | 있다 |
| 食べる/飲む | 召し上がる(めしあがる) | いただく | 먹다/마시다 |
| 言う | おっしゃる | 申す(もうす) | 말하다 |
| する | なさる | いたす | 하다 |
| 見る | ご覧になる(ごらんになる) | 拝見する(はいけんする) | 보다 |
| 知る | ご存じ(ごぞんじ) | 存じる(ぞんじる) | 알다 |

## 발음 주의 사항

### 한국어에 없는 일본어 발음

| 일본어 | 한국어 근사 | 발음 팁 |
|---|---|---|
| つ(tsu) | 쯔/츠 | "ㅊ"보다 혀끝을 윗니 뒤에 더 가깝게 |
| ふ(fu) | 후 | 영어 f가 아니라 양 입술 사이로 바람 |
| ら행(ra,ri,ru,re,ro) | 라리루레로 | 영어 r이 아닌 혀끝 가볍게 튕기기 (탄음) |
| ん(n) | ㄴ/ㅁ/ㅇ | 위치에 따라 변화 (な행 앞=ㄴ, ま행 앞=ㅁ, か행 앞=ㅇ) |

### 장음 규칙

- あ단 + あ → 장음: おかあさん (오카-상, 어머니)
- い단 + い → 장음: おにいさん (오니-상, 형/오빠)
- う단 + う → 장음: くうき (쿠-키, 공기)
- え단 + い → 장음: せんせい (센세-, 선생님) ※ 에이가 아니라 에-로 발음
- お단 + う → 장음: おとうさん (오토-상, 아버지) ※ 오우가 아니라 오-로 발음

### 악센트 (높낮이)

- 일본어는 **고저 악센트** (pitch accent), 한국어와 유사
- 영어의 강세와 다르게 높낮이로 구분
- 예: はし → 箸(젓가락: 높+낮) vs 橋(다리: 낮+높)
- 표준어(도쿄 방언) 기준, 지역별 차이 있음
