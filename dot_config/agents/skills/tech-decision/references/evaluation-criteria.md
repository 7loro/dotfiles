## 신뢰도 평가 기준

정보 출처별 기본 신뢰도:

| 출처 | 신뢰도 | 비고 |
|------|--------|------|
| 공식 문서 (Context7) | 높음 (90~100%) | 가장 신뢰할 수 있는 1차 출처 |
| 독립 벤치마크 | 높음 (80~95%) | 재현 가능한 테스트 조건 확인 필요 |
| GitHub Issues/Discussions | 중간~높음 (70~85%) | 실제 사용자 문제 반영 |
| Stack Overflow | 중간 (60~75%) | 답변 날짜와 투표 수 확인 |
| Reddit / Hacker News | 중간 (50~70%) | 개인 경험 편향 가능 |
| 블로그 / 튜토리얼 | 낮음~중간 (40~60%) | 작성자 전문성과 날짜 확인 |
| 마케팅 자료 | 낮음 (20~40%) | 긍정 편향 고려 |

### 신뢰도 보정 요인

- **최신성**: 1년 이내 +10%, 3년 이상 -20%
- **다수 출처 일치**: 3개 이상 독립 출처 일치 시 +15%
- **메인테이너 발언**: 공식 채널 발언 시 신뢰도 상향
- **재현 가능성**: 벤치마크 코드 공개 시 +10%
